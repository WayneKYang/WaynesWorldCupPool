⚙️ One-Time Setup for Cross-Device Sync
Step 1. Go to script.google.com and click New project
Step 2. Delete the default code. Paste the code from the Apps Script file you downloaded alongside this HTML file.
Step 3. Click Deploy → New deployment → Web app
Step 4. Set Execute as: Me · Set Who has access: Anyone · Click Deploy
Step 5. Copy the web app URL → paste it in Admin → Cloud Sync Setup → Save
Step 6. Done! All devices now sync automatically.
